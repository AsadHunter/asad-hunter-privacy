Privacy Policy for Asad Hunter

Asad Hunter is a Chrome extension designed to help marketers, advertisers, and researchers analyze publicly available advertising transparency data from the Meta (Facebook & Instagram) Ad Library.

Information Collection

Asad Hunter does not collect, store, sell, or share any personal user data. The extension only processes publicly visible information displayed on the Meta Ad Library website when the user explicitly interacts with it.

Permissions Usage

  • activeTab: Used only when the user clicks the extension to analyze the currently opened Meta Ad Library page.
  • Host permissions: Required to access and analyze publicly available ad transparency data on facebook.com/ads/library.
  • Downloads: Used only when the user explicitly chooses to download available ad creatives or related data.

Third-Party Sharing

Asad Hunter does not share any personal or sensitive user data with third parties. All analysis is performed locally or using publicly available data sources.

User Control

All functionality is fully user-initiated. The extension does not perform background tracking, monitor browsing behavior, or operate on websites other than the supported Meta Ad Library.

Data Security

Asad Hunter does not store user data. Since no personal data is collected, there is no risk of data retention or unauthorized access.

Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Any updates will be reflected on this page.
